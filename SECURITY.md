# Security Policy

## Supported Versions

Supported .NET Versions for this library.

| Version | Supported          |
| ------- | ------------------ |
| 6.0.x   | :white_check_mark: |
| < 6.0   | :x:                |

## Reporting a Vulnerability

If you found an issue or a vulnerability, open an issue with the most specific ways to replicate the issue.
